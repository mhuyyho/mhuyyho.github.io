# DLL SideLoading

**DLL SideLoading** là một kỹ thuật tấn công phổ biến trong lĩnh vực an ninh mạng, cho phép kẻ tấn công thực thi mã độc bằng cách lợi dụng cách hệ điều hành Windows tải các thư viện DLL.
